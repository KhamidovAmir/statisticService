# statisticService
Проект является учебным и выполняет функции хранения, обработки различных статистических данных.
В качестве базы данных используется Clickhouse, данные предположительно должны приходить из очереди RabbitMQ.
Сервис является частью одного большого учебного проекта с микросервисной архитектурой.
Написан проект только на Java с использованием фреймворка Spring
